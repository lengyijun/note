# 各大tiling window的测评

## awesome

没有主从窗口思想.窗口地位平等

### 优点

- 增加了window置顶这些功能

### 缺点

- 标题不能缩到bar里面,浪费了一行
- bar是自成体系,不能利用polybar这些
- 配置有点复杂
- 默认启动器没有dmenu好用5

------

## DWM

主从窗口思想.主窗口放在左边,副窗口自动缩小

### 优点

- 标题能缩到bar里面

### 缺点

- 配置格外复杂
- 只有两种窗口布局,一种浮动,一种左右.上下做不到

------

## Leftwm

### 优点

- 配置用toml,很不错
- 标题能缩到bar里面

### 缺点

- workspace非常混乱
